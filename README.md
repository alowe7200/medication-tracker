# Medication Tracker

A lightweight health tracking app for logging daily symptoms, sleep, digestion, bloating, breathing, and overall wellness.

Built to make daily tracking simple and fast for non-technical users.

## Live App

https://medication-tracker-zeta.vercel.app

## Features

- Daily health ratings (emoji-based)
- Habit tracking
- Medication tracking
- Simple metrics dashboard
- LocalStorage persistence (no account required)

## Tech Stack

- React
- Vite
- JavaScript
- CSS
- Vercel (deployment)
